01 · Tipo de gráfica

Elige el tipo correcto

La pregunta que quieres responder determina el tipo de gráfica.

¿Cómo cambia Y en el tiempo?

Línea

¿Cómo se comparan categorías?

Barras

¿Existe correlación entre X e Y?

Dispersión

¿Cómo se distribuyen los datos?

Box plot

En sus proyectos: dispersión para correlaciones, líneas para cinéticas y curvas H–Q, barras para comparar tratamientos.

01 · Tipo incorrecto

Pie con muchas categorías

¿Por qué falla?

  • Los ángulos engañan al ojo humano.
  • Difícil comparar segmentos similares.
  • 8 categorías son demasiadas.
  • Los colores no comunican nada.

01 · Tipo correcto

Barras horizontales, ordenadas

¿Por qué funciona?

  • Longitudes fáciles de comparar.
  • Ordenado de mayor a menor.
  • Un solo color (misma variable).
  • Valores exactos visibles.
  • Etiquetas legibles.

02 · Ratio datos-tinta

"Cada gota de tinta debe tener una razón de existir."— Edward Tufte

Eliminar

  • Fondos grises o coloreados.
  • Gridlines excesivos.
  • Bordes innecesarios.
  • Leyendas redundantes.
  • 3D (siempre).
  • Efectos decorativos.

Mantener

  • Los datos (protagonistas).
  • Etiquetas de ejes (contexto).
  • Anotaciones que guían.
  • Título argumentativo.
  • Una leyenda si es necesaria.
  • Color con propósito.

04 · Color con propósito

Tres familias de paletas

Cualitativa

Distinguir categorías sin orden.

Tratamientos, grupos experimentales.

Secuencial

Representar magnitud (bajo → alto).

Temperatura, concentración, tiempo.

Divergente

Desviaciones de un centro.

Error vs. referencia, cambio %.

Accesibilidad: evitar combinaciones rojo-verde. Preferir paletas como viridis o cividis.

Herramientas

Con qué construir las gráficas

Excel bien usado

Eliminar elementos por defecto. Ajustar colores, tipografía, ejes. Un makeover transforma la gráfica.

Python (matplotlib)

Control total. Reproducible. Ideal cuando se generan muchas gráficas del mismo tipo.

IA generativa

Describir lo que necesitas en lenguaje natural con ChatGPT o Claude. La IA genera el código. Tú decides el mensaje.

Complementarias

Canva, BioRender, draw.io para diagramas de proceso y esquemas experimentales.

La escala AIAS permite Nivel 3 (IA para edición) en el proyecto.

Ejemplo

Prompt para visualización con IA

Tengo datos de viscosidad (cP) vs temperatura (°C) para 3 concentraciones de polímero (1 %, 3 %, 5 %). Genera un gráfico de líneas en Python con: - Fondo blanco, sin bordes - Paleta: azul oscuro, dorado, gris - Gridlines solo horizontales, sutiles - Título argumentativo sobre el hallazgo - Leyenda integrada - Accesible para daltonismo
El estudiante define qué comunicar. La IA implementa cómo.
